== Description ==

Small plugin create to help store owner add gift wrapping fee to order and let buyer select a gift card if they want

Require WooCommerce version 2.0 or later

== Installation ==

Just download and upload all file into plugins folder or install with original zip file.

== Frequently Asked Questions ==
= Plugin installed and active but can not see setting? =

You can found it at WooCommerce Settings -> Products -> NA Gift Wrapping at end
